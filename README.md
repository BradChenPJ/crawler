# crawler
Crawl several brands' information in Taiwan and public construction information. Export a CSV after crawling.
## Contents
* tw_pcc folder: Run `main.py` to execute crawler. You can get three outputs.
  * smallCounty.csv
  * engineering.csv
  * metadata.csv
* twn_carrefour.py
* twn_domino.py
* twn_familymart.py
* twn_ftc.py
* twn_kfc.py
* twn_mcdonald.py
* twn_pxmart.py
